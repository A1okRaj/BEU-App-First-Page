# 🎓 BEU Student App

A student-focused mobile application for Bihar Engineering University (BEU) students.  
Currently building this project using React Native & Expo to simplify student academic life.

---

## 🚀 Planned Features

- 📊 SGPA / CGPA Calculator
- 📚 Notes & PYQ Section
- 📝 Result Dashboard
- 📅 Attendance Tracker
- 🔔 Important Notifications
- 🌙 Modern UI/UX
- ⚡ Fast & Smooth Performance

---

## 🛠️ Tech Stack

- React Native
- Expo
- JavaScript
- React Navigation

---

## 🎯 Vision

To create an all-in-one platform for BEU students where they can easily manage academics, track progress, and access important resources.
